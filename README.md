git-back-checkout
===============

Installation
------------

```sh
go get github.com/pocke/git-back-checkout
```

Usage
--------

```sh
$ git-back-checkout <Branch Name> [<Revision>]

# e.g.
$ git-back-checkout feature/foobar
$ git-back-checkout feature/bazbaz HEAD~~
```
